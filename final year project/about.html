About Smart Agro

Deep Learning Based Pesticides Recommendation Framework for Plant Disease Control

🎯 Project Overview

Final year project using a Convolutional Neural Network (CNN) with 39 output classes trained on the PlantVillage dataset. Upload a 224×224 leaf image to detect disease and get pesticide advice.

💻 Technology

  • Deep Learning: PyTorch CNN (Conv2d → BatchNorm → MaxPool ×4 → Linear 1024 → 39)
  • Backend: Flask (Python)
  • Frontend: HTML, CSS, JavaScript — responsive design
  • Parameters: 52,595,399 trainable · Input 224×224 RGB
